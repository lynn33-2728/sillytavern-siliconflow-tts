# Notice

This project is a modified version of:

- Original repository: https://github.com/hjl2004-10/extension
- Original author: hjl2004-10

Changes in this version focus on the floating audio progress bar for SillyTavern:

- Replaced the visible native browser audio controls with a custom floating player UI.
- Added persistent show/hide behavior controlled by a settings toggle.
- Added left-bottom default placement for small browser windows, tablets, and phones.
- Added menu items for downloading the latest generated audio, playback speed, logs, and position reset.
